# RASA-Chatbot-UI
A Chat widget easy to connect to RASA core bot to Custom Channel.

Added Support for 
- Buttons
- Images

Note: It works only with Rest Input Channel, for more details head on to http://rasa.com/docs/rasa/user-guide/connectors/custom-connectors/
